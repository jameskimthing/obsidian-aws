# Trusted Advisor
-  Automatically performs some predefined checks && offers recommendations
	- Recommendations that can help improve:
		- Cost optimization
		- Performance
		- Security
		- Fault tolerance
		- Service quotas
- Provides additional checks via higher-level [[Support#AWS Support Plans|Support Plan]]s
